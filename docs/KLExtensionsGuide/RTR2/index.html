RTR2 Extension¶

Fabric Engine version 2.3.0
Copyright (c) 2010-2016, Fabric Software Inc. All rights reserved.

The RTR2 extension provides multiple components that can be assembled for building a customized rendering pipeline. The RTR’s goal is to provide a good balance between performance and flexibility.

See RealTime Renderer for an overview of the framework.

See SceneHub for a global picture of the SceneGraph, the Scene Assembly and the RealTime Renderer.

注釈

Various RTR samples are located in the /Samples/RTR2 and can be run with the sceheHub app, for example sceneHub /Samples/RTR2/MovingHierarchy.kl (use the `` –unguarded`` flag for optimal performance but unguarded KL compilation).
