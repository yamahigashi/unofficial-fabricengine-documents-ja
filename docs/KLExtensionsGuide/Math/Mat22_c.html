Mat22_c (struct)¶

The Mat22_c represents a 2 by 2 matrix. It uses Vec2_c types for each of the 2 rows.

/*
** Example: Construct Mat22_c
*/

// create a matrix 2x2
Vec2_c v0(1.0, 0.0);
Vec2_c v1(0.0, 1.0);
Mat22_c mat(v0, v1);
