FileSystem (struct)¶

バージョン 1.12.0 で追加.

The FileSystem maps closely to the boost::filesystem static methods. Please see the boost documentation for each of the methods.

http://www.boost.org/doc/libs/1_55_0/libs/filesystem/doc/operations.htm
