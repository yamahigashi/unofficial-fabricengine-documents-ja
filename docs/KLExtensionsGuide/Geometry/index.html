Geometry エクステンション¶

Fabric Engine version 2.3.0
Copyright (c) 2010-2016, Fabric Software Inc. All rights reserved.

Geometry のエクステンションは、例えば Points,や Lines, PolygonMesh といった一般的なジオメトリ型のKLでの実装を提供します。エクステンションは、これらジオメトリ型に共通する様々なコンポーネントやインターフェースを定義します。

全てのジオメトリ型は Geometry インターフェースを実装しています。それらは、ポジションやUV、カラーといったコンポーネントに関連付けされているアトリビュート値を保持するために、 GeometryAttributes という単一のコンテナ（データ型）を使用します。これには、 Vec3Attributes や ColorAttributes といった様々なアトリビュート型があります。 GeometryAttribute インターフェースが定義付けされている限り、カスタムした様々な型のアトリビュートをジオメトリに結合することが可能です。

全てのジオメトリ型は、レイキャストや最接近点の取得といった空間分析を実行することが可能になる SpatialQueryable インターフェースを実装しています。これは空間分析の速度を上げるためのジオメトリを内部で使用した Octree と SparseGrid という2つの高速化構造を定義します。このインターフェースについての詳細は ジオメトリを用いた空間分析群(Spatial queries)の使用 を参照してください。

.

ジオメトリは、処理をGPU上で行うことで高いパフォーマンスを可能にするGPUコンピュートカーネルを利用することが出来ます。既存のコードは最小限の変更によってGPU上で実行できるようになります。GPUでジオメトリを使用する方法の詳細については、 utilizinggpucompute を参照してください。
