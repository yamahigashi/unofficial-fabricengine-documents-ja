Characters エクステンション¶

Fabric Engine version 2.3.0
Copyright (c) 2010-2016, Fabric Software Inc. All rights reserved.

キャラクターエクステンションは、群衆やプリビズ用ツールのようなハイレベルなキャラクターツール構築のための基本のクラス群とインターフェース群を提供します。基本のキャラクター型はカスタムキャラクター定義をロードしたりファブリックのレンダリングツールを使用してそれらをレンダーするツール作成のために、拡張と特化が可能です。
